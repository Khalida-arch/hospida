## Key lessons learned
- Google Sheets is not the right database for a real product
  — formulas don't auto-populate on new rows without 
  ArrayFormula, and ArrayFormula caused its own errors
- Supabase solves all of this cleanly — proper database columns,
  no formulas needed, real-time built in
- Always confirm the exact field path in Make's webhook output 
  before mapping — don't guess
- The transcript already contains all the structured data Soji 
  collected — location, type, condition are all in there
- Replay Run in Make saves time — no need for a new test call 
  when debugging mappings
